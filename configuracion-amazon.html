Configuración de categorías Amazon

Esta página sirve como acceso interno para revisar categorías, keywords, ASINs prioritarios y mejora continua. No guarda claves privadas ni credenciales.

Regla de seguridad

No pongas Access Key, Secret Key, tokens ni credenciales de Amazon en HTML público. Las claves deben estar en servidor, variables de entorno o GitHub Secrets.

Categorías configuradas

1. Método recomendado

Usar Amazon Product Advertising API 5.0 con SearchItems para búsquedas por keyword y GetItems para ASINs concretos.

2. Método inicial

Crear enlaces afiliados manuales por categoría hasta tener acceso estable a API. Mantiene el SEO y permite validar conversión.

3. Mejora continua

Revisar semanalmente clics, CTR, productos sin stock, consultas y categorías con más intención de compra.
